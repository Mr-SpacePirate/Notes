## 0x53 $Element$

### 环境配置

- 安装 ElementUI 组件库（在当前工程的目录下），在命令行执行指令：
  ```cmd
  npm install element-ui@版本号
  ```
- 引入 ElementUI 组件库

  ```js
  // main.js
  import ElementUI from "element-ui";
  import "element-ui/lib/theme-chalk/index.css";

  Vue.use(ElementUI);
  ```
