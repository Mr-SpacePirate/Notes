## 0x47 方差公式

$$
s^2 = \frac{\sum^{n}_{i = 1} (x - \bar{x}) ^ 2}{n} = \frac{1}{n} \sum ^{n} _{i = 1} x_i^2 - \bar{x} ^2 
$$

**推论**：平方和区间每个数加上 $a$
$$
x_1^2 + x_2^2 + \dots + x_n^2 \rightarrow (x_1 + a)^2 + (x_2 + a)^2 + \dots + (x_n + a)^2
$$
$$
\rightarrow x_1^2 + x_2^2 + \dots + x_n^2 + 2a(x_1 + x_2 + \dots + x_n) + na^2
$$
